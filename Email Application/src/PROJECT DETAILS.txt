Project 1
Email Application

by lcastaa (Luis Castaneda)

[Scenario] -
I'm an IT Support Administrator Specialist, and I am Tasked with creating email accounts
for new hires.

[My Application Should do the following] -
    - Generate an email with the following syntax (firstname, lastname@department.company.com)
    - Determine the Department(Sales, development, accounting, human resources), if none leave blank
    - Generate a random String for a password
    - Have Set Methods to Change the password, set mailbox capacity , and define an alternate email address.
    - Have Methods display name, email, and capacity.
